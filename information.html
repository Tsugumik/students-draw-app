Information on the processing of personal data and the terms of use

The "Student Draw" application stores all the data on the user's device.

The user of the application is solely responsible for the security and integrity of the data entered in the forms.

The code is entirely executed on the client's side. No personal data is transferred off the user's device.



The application uses the random.org API to generate truly random numbers.
No personal data of students or others stored in the browser's local storage is sent to random.org.



By using the application, the user accepts the terms and conditions of the API random.org available at https://www.random.org/terms/2020-08-01.
